# Web de Propinas y Reseñas

Landing estilo Linktree para compartir:

- Botones rápidos de contacto/acciones.
- Datos bancarios para propina.
- QR para escanear y pagar.
- Enlace directo para reseñar al restaurante en Google.

## Personalización rápida

1. Edita `script.js` y reemplaza los valores del objeto `SITE_CONFIG`.
2. Cambia el QR de ejemplo en `assets/qr-propina.svg` por tu QR real (o usa otro archivo y actualiza la ruta en `SITE_CONFIG.tip.qrImage`).
3. Abre `index.html` en el navegador.

## Publicación

Puedes subir estos archivos a Netlify, Vercel, GitHub Pages o cualquier hosting estático.
